# Contributing to react-tailwindcss-debugger

First off, thank you for considering contributing to `react-tailwindcss-debugger`! It's people like you that make the open-source community such an amazing place.

## How Can I Contribute?

### Reporting Bugs
If you find a bug, please search the [issues](https://github.com/AnkitCHAKRABORTY0510/react-tailwindcss-debugger/issues) to see if it has already been reported. If not, feel free to open a new issue with a clear description and steps to reproduce.

### Suggesting Enhancements
We're always looking for ways to make the debugger better! If you have an idea, please open an issue to discuss it.

### Pull Requests
1. **Fork** the repository and create your branch from `main`.
2. **Install** dependencies: `npm install`.
3. **Make** your changes.
4. **Ensure** your code follows the existing style.
5. **Test** your changes locally.
6. **Submit** a Pull Request!

## Development Setup

1. Clone your fork:
   ```bash
   git clone https://github.com/YOUR_USERNAME/react-tailwindcss-debugger.git
   cd react-tailwindcss-debugger
   ```
2. Install dependencies:
   ```bash
   npm install
   ```
3. Run in development mode:
   ```bash
   npm run dev
   ```
